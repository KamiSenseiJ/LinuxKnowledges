# 1. man command

1. introduce: man, through which, we can search all linux command's reference document;
1. usage: `man xxx_command`
1. basic operaions
    1. h: operations;
    1. up, down, left, right arrow: moving around by one line or column;
    1. pageup, pagedown: moving page;
    1. g: to document head; shift + g: to document tail;
    1. search
        - input slash ('/'), then input search string and enter; 
        - n for next match;
        - shift + n for previous match;
    1. q: quit document;
